Install Flask Python ReactJS:
=============================
#Install prerequisites
sudo apt update && sudo apt upgrade
sudo apt install python3.10-venv
sudo apt install curl
sudo apt install gh
sudo apt install geany
cd ~
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
#Close and re-open terminal
source ~/.bashrc
nvm install v18.1.0 #Or use another version listed if need be
sudo npm install --global yarn
#Test all
node --version
npm --version
yarn --version
python3 --version
#Configure global git
git config --global user.name "Your_Name"
#git config --global user.email "email_address@domain.com" #Avoid if not required...

Install frameworks
===================
#Create React/Flask app
npx create-react-app react-flask-app
cd react-flask-app
mkdir api
cd api
python3 -m venv venv
source venv/bin/activate #To switch to virtual python env. (do so before installing any package via pip from then on)
pip install flask python-dotenv

#Then coding etc. tutorial on : https://blog.miguelgrinberg.com/post/how-to-create-a-react--flask-project
#Or: https://fr.reactjs.org/tutorial/tutorial.html
